# React App

Bu, Infrawatch monorepo'sunda yer alan düz React uygulamasıdır.

## Teknolojiler

- React 18
- TypeScript
- Vite
- Shared UI Components (@repo/ui)

## Geliştirme

```bash
# Bağımlılıkları yükle
pnpm install

# Geliştirme sunucusunu başlat
pnpm dev

# Build oluştur
pnpm build

# Build'i önizle
pnpm preview

# Linting
pnpm lint

# Type checking
pnpm check-types
```

## Port

Bu uygulama `http://localhost:3002` adresinde çalışır.
